# Sociedad Digital

**Tarea 8 — Diseña una web**
Autor: **Mohit Deepak Soni** · Grupo: **2º de Bachillerato**

Sitio web de una sola página que resume cuatro conceptos clave sobre nuestra
vida en la era digital, con enlaces de ampliación, reflexiones personales y un
apartado que explica cómo se ha construido el propio sitio.

## 🌐 Ver el sitio publicado

Una vez activado GitHub Pages, el sitio estará disponible en:

```
https://mohitdeepaksoni.github.io/sociedad-digital/
```

## 📋 Contenido

El sitio cubre los cuatro temas pedidos en el enunciado:

1. **El sector del desarrollo de software** — su impacto en la sociedad
   (empleo, innovación, desigualdad y cambio cultural).
2. **El comercio electrónico** — tipos de plataformas (B2C, B2B, C2C, D2C,
   suscripción), estrategias de venta y seguridad en las transacciones.
3. **La ciberseguridad en los medios digitales** — phishing, robo de
   identidad, fraude, malware e ingeniería social, y cómo protegerse.
4. **La ciudadanía digital crítica** — protección de datos (RGPD), ética en
   redes, publicidad responsable y comercio justo.

Además incluye una sección de **reflexiones personales** sobre cada tema y un
**colofón** que documenta el proceso de creación.

## 🛠️ Cómo está hecho

- **HTML y CSS** escritos a mano (sin frameworks ni plugins).
- **Tipografías:** Fraunces, Manrope y JetBrains Mono (Google Fonts).
- **Diseño:** estilo editorial tipo revista, con variables CSS para mantener
  una paleta coherente (papel, tinta y un acento bermellón).
- **Responsive:** se adapta a móvil y escritorio.
- **Sitio estático:** un único archivo `index.html`, sin bases de datos.

## 📂 Estructura del repositorio

```
sociedad-digital/
├── index.html      # El sitio web completo
└── README.md       # Este archivo
```

## 🚀 Cómo activar GitHub Pages

1. En este repositorio, ve a **Settings** → **Pages**.
2. En *Source*, elige **Deploy from a branch**.
3. Selecciona la rama **main** y la carpeta **/ (root)**.
4. Pulsa **Save** y espera 1–2 minutos.
5. La URL pública aparecerá en la parte superior de esa misma página.
